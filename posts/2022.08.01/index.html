3.1 Three.js

Three.js란? 웹 브라우저에서 WebGL을 통해 3차원 컴퓨터 그래픽스 애니메이션 응용을 만들고 표현하기 위해 사용되는 Javascript Library이자, API.

WebGL(Web Graphics Library): 웹상에서 2D 및 3D 그래픽을 렌더링하기 위한 Javascript API -> HTML5 Canvas 요소 위에 렌더링됨

Three.js의 구조


3.1.1 Renderer

  • Three.js의 핵심 객체로 Scene과 Camera 객체를 넘겨 받아 카메라의 절두체 안 3D안 씬의 일부를 평면(2D) 이미지로 렌더링함


3.1.2 Scene

  • Object가 Render되는 장소이다 - 물체(Object), 빛(Light) 등을 놓는 장소
  • Scene에 Object, Light 등을 추가하고 카메라와 같이 Render한다.

3.1.3 Camera

Camera: 사물을 비춰주는 렌즈 역할을 하는 클래스


기본 요소

PerspectiveCamera(fov: number, aspect: number, near: number, far : number);

  • fov: 기본값50(45~75)
  • aspect: 기본값1(canvas width/canvas height)
  • near: 기본값0.1
  • far: 기본값2000
